## About Borebase

**Borebase** is a cloud platform built for geotechnical and geological companies. It brings project management, data storage, analysis, and report generation into one place - so your team can focus on engineering, not spreadsheets.

Whether you manage a single site investigation or hundreds of projects across the country, Borebase gives you the tools to organize, analyze, and deliver results faster.

<br/>

## What You Can Do

<table>
<tr>
<td width="50%" valign="top">

### Manage Projects on a Map
Create and organize geotechnical projects with interactive maps. Pin your boreholes, CPTs, and SPTs directly on the project site. Navigate between objects visually.

### Store All Your Data in the Cloud
No more scattered files. Upload, structure, and access your borehole logs, CPT soundings, SPT results, and geophysical surveys from anywhere. Your entire team sees the same data in real time.

</td>
<td width="50%" valign="top">

### Analyze CPT, SPT & Borehole Data
Run industry-standard calculations automatically: Robertson classification, ISO 22476-1 parameters, SPT correlations, and more. View depth-based graphs for qc, fs, Rf, u2, qt, N60, Ic, tilt, and speed.

### Generate Professional Reports
Build multi-column PDF exports with drag-and-drop. Combine stratigraphy hatches, graphs, and data from different objects on a single page. Save layouts and reuse them across projects.

</td>
</tr>
</table>
